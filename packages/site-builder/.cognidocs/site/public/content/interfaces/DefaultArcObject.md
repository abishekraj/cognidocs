# DefaultArcObject

Interface corresponding to the minimum data type assumed by the accessor functions of the Arc generator.

## Properties

| Name          | Type                  | Optional | Description |
| :------------ | :-------------------- | :------- | :---------- |
| `innerRadius` | `number`              | No       | -           |
| `outerRadius` | `number`              | No       | -           |
| `startAngle`  | `number`              | No       | -           |
| `endAngle`    | `number`              | No       | -           |
| `padAngle`    | `number \| undefined` | Yes      | -           |
