# SankeyDiagramConfig

The object containing configurations specific for sankey diagrams. This interface was referenced by `MermaidConfig`'s JSON-Schema via the `definition` "SankeyDiagramConfig".

## Properties
| Name | Type | Optional | Description |
| :--- | :--- | :------- | :---------- |
| `width` | `number` | Yes | - |
| `height` | `number` | Yes | - |
| `linkColor` | `SankeyLinkColor \| string` | Yes | - |
| `nodeAlignment` | `SankeyNodeAlignment` | Yes | - |
| `useMaxWidth` | `boolean` | Yes | - |
| `showValues` | `boolean` | Yes | - |
| `prefix` | `string` | Yes | - |
| `suffix` | `string` | Yes | - |
