# IconifyMetaData

Meta data stored in JSON file, used for browsing icon set.

## Properties

| Name         | Type                     | Optional | Description |
| :----------- | :----------------------- | :------- | :---------- |
| `info`       | `IconifyInfo`            | Yes      | -           |
| `chars`      | `IconifyChars`           | Yes      | -           |
| `categories` | `IconifyCategories`      | Yes      | -           |
| `themes`     | `LegacyIconifyThemes`    | Yes      | -           |
| `prefixes`   | `Record<string, string>` | Yes      | -           |
| `suffixes`   | `Record<string, string>` | Yes      | -           |
