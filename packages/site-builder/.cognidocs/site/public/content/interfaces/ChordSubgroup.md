# ChordSubgroup

A chord subgroup serving as source or target of a chord between two nodes i an j (where i may be equal to j).

## Properties

| Name         | Type     | Optional | Description |
| :----------- | :------- | :------- | :---------- |
| `startAngle` | `number` | No       | -           |
| `endAngle`   | `number` | No       | -           |
| `value`      | `number` | No       | -           |
| `index`      | `number` | No       | -           |
