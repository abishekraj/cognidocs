# IconifyJSONPackageExports

Structure of exports '@iconify-json/*' packages. These are small packages, one per icon set, that split JSON structure into multiple files to reduce amount of data imported from package.

## Properties
| Name | Type | Optional | Description |
| :--- | :--- | :------- | :---------- |
| `info` | `IconifyInfo` | No | - |
| `icons` | `IconifyJSON` | No | - |
| `metadata` | `IconifyMetaData` | No | - |
| `chars` | `IconifyChars` | No | - |
