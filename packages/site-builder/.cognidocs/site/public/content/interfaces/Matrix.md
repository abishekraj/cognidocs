# Matrix

Administrivia: a matrix of numeric values. If height is not specified, it is inferred from the given width and data.length.

## Properties

| Name     | Type                | Optional | Description |
| :------- | :------------------ | :------- | :---------- |
| `data`   | `ArrayLike<number>` | No       | -           |
| `width`  | `number`            | No       | -           |
| `height` | `number`            | Yes      | -           |
