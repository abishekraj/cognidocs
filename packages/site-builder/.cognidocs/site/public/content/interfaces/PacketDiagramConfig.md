# PacketDiagramConfig

The object containing configurations specific for packet diagrams. This interface was referenced by `MermaidConfig`'s JSON-Schema via the `definition` "PacketDiagramConfig".

## Properties
| Name | Type | Optional | Description |
| :--- | :--- | :------- | :---------- |
| `rowHeight` | `number` | Yes | - |
| `bitWidth` | `number` | Yes | - |
| `bitsPerRow` | `number` | Yes | - |
| `showBits` | `boolean` | Yes | - |
| `paddingX` | `number` | Yes | - |
| `paddingY` | `number` | Yes | - |
