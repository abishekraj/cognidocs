# ClientPointEvent

A User interface event (e.g. mouse event, touch or MSGestureEvent) with captured clientX and clientY properties.

## Properties

| Name      | Type     | Optional | Description |
| :-------- | :------- | :------- | :---------- |
| `clientX` | `number` | No       | -           |
| `clientY` | `number` | No       | -           |
