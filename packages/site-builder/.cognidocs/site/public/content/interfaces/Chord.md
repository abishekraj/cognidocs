# Chord

A chord represents the combined bidirectional flow between two nodes i and j (where i may be equal to j)

## Properties

| Name     | Type            | Optional | Description |
| :------- | :-------------- | :------- | :---------- |
| `source` | `ChordSubgroup` | No       | -           |
| `target` | `ChordSubgroup` | No       | -           |
