# Feature

A feature object which contains a geometry and associated properties. https://tools.ietf.org/html/rfc7946#section-3.2

## Properties

| Name         | Type                            | Optional | Description |
| :----------- | :------------------------------ | :------- | :---------- |
| `type`       | `"Feature"`                     | No       | -           |
| `geometry`   | `G`                             | No       | -           |
| `id`         | `string \| number \| undefined` | Yes      | -           |
| `properties` | `P`                             | No       | -           |
