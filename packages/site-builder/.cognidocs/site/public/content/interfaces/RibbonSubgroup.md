# RibbonSubgroup

A minimal interface to support the default accessors used by RibbonGenerator for properties of source and target objects of a Ribbon. (Corresponds to ChordSubgroup)

## Properties
| Name | Type | Optional | Description |
| :--- | :--- | :------- | :---------- |
| `startAngle` | `number` | No | - |
| `endAngle` | `number` | No | - |
| `radius` | `number` | No | - |
