# DocumentLink

A document link is a range in a text document that links to an internal or external resource, like another text document or a web site.

## Properties
| Name | Type | Optional | Description |
| :--- | :--- | :------- | :---------- |
| `range` | `Range` | No | - |
| `target` | `URI` | Yes | - |
| `tooltip` | `string` | Yes | - |
| `data` | `LSPAny` | Yes | - |
