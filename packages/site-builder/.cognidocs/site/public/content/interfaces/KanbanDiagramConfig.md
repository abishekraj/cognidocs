# KanbanDiagramConfig

The object containing configurations specific for kanban diagrams This interface was referenced by `MermaidConfig`'s JSON-Schema via the `definition` "KanbanDiagramConfig".

| `padding`       | `number` | Yes      | -           |
| `sectionWidth`  | `number` | Yes      | -           |
| `ticketBaseUrl` | `string` | Yes      | -           |
