# CstNodeLocation

The Column/Line properties will only be present when The {@link IParserConfig.nodeLocationTracking} is set to "full".

## Properties
| Name | Type | Optional | Description |
| :--- | :--- | :------- | :---------- |
| `startOffset` | `number` | No | - |
| `startLine` | `number` | Yes | - |
| `startColumn` | `number` | Yes | - |
| `endOffset` | `number` | Yes | - |
| `endLine` | `number` | Yes | - |
| `endColumn` | `number` | Yes | - |
