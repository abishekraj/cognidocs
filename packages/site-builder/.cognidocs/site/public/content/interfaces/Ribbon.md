# Ribbon

A minimal interface to support the default source and target accessors used by RibbonGenerator. (Corresponds to Chord)

## Properties

| Name     | Type             | Optional | Description |
| :------- | :--------------- | :------- | :---------- |
| `source` | `RibbonSubgroup` | No       | -           |
| `target` | `RibbonSubgroup` | No       | -           |
