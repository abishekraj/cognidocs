# ExtendedFeatureCollection

A GeoJSON-style FeatureCollection which supports GeoJSON features and features built on GeoSphere The generic refers to the type of ExtendedFeature contained in the ExtendedFeatureCollection.

## Properties

| Name       | Type            | Optional | Description |
| :--------- | :-------------- | :------- | :---------- |
| `features` | `FeatureType[]` | No       | -           |
