# parseJSDoc

**Return Type:** `JSDocComment`

## Parameters

| Name      | Type                | Optional | Description                                                                  |
| :-------- | :------------------ | :------- | :--------------------------------------------------------------------------- |
| `node`    | `CstNode`           | No       | A `CstNode` from a parsed Langium document.                                  |
| `options` | `JSDocParseOptions` | Yes      | Parsing options specialized to your language. See {@link JSDocParseOptions}. |
