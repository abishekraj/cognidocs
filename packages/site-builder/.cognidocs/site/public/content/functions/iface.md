# iface

Defines an interface. The first argument is an array of interfaces that it extends, and the second is an array of properties.

**Return Type:** `TIface`

## Parameters

| Name                                   | Type       | Optional | Description |
| :------------------------------------- | :--------- | :------- | :---------- |
| `bases`                                | `string[]` | No       | -           |
| `props`                                | `{         |
| [name: string]: TOptional \| TypeSpec; |
| }`                                     | No         | -        |
