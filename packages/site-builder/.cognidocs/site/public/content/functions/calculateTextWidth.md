# calculateTextWidth

This calculates the width of the given text, font size and family.

**Return Type:** `ReturnType<typeof calculateTextDimensions>['width']`

## Parameters

| Name     | Type                                            | Optional | Description                                                                          |
| :------- | :---------------------------------------------- | :------- | :----------------------------------------------------------------------------------- |
| `text`   | `Parameters<typeof calculateTextDimensions>[0]` | No       | The text to calculate the width of                                                   |
| `config` | `Parameters<typeof calculateTextDimensions>[1]` | No       | The config for fontSize, fontFamily, and fontWeight all impacting the resulting size |
