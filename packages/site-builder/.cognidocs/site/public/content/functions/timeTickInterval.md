# timeTickInterval

Returns the time interval that would be used by d3.timeTicks given the same arguments.


**Return Type:** `TimeInterval | null`

## Parameters
| Name | Type | Optional | Description |
| :--- | :--- | :------- | :---------- |
| `start` | `Date` | No | - |
| `stop` | `Date` | No | - |
| `count` | `number` | No | - |
