# expectNoErrorsOrAlternatives

A helper test function that validate that the result doesn't have errors or any ambiguous alternatives from chevrotain.

**Return Type:** `void`

## Parameters

| Name     | Type          | Optional | Description                  |
| :------- | :------------ | :------- | :--------------------------- |
| `result` | `ParseResult` | No       | the result `parse` function. |
