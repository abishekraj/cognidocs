# fromMarkdown

Turn markdown into a syntax tree.

**Return Type:** `Root`

## Parameters

| Name       | Type                            | Optional | Description        |
| :--------- | :------------------------------ | :------- | :----------------- |
| `value`    | `Value`                         | No       | Markdown to parse. |
| `encoding` | `Encoding \| null \| undefined` | Yes      | -                  |
| `options`  | `Options \| null \| undefined`  | Yes      | -                  |
