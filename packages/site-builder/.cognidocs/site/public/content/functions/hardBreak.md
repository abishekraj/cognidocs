# hardBreak

Turn an mdast `break` node into hast.

**Return Type:** `Array<Element | Text>`

## Parameters

| Name    | Type    | Optional | Description         |
| :------ | :------ | :------- | :------------------ |
| `state` | `State` | No       | Info passed around. |
| `node`  | `Break` | No       | mdast node.         |
