# addEdges

Add edges to cytoscape instance from provided edge array This function processes only the edges provided in the data structure

**Return Type:** `void`

## Parameters

| Name    | Type             | Optional | Description            |
| :------ | :--------------- | :------- | :--------------------- |
| `edges` | `Edge[]`         | No       | Array of edges to add  |
| `cy`    | `cytoscape.Core` | No       | The cytoscape instance |
