# stringifyEntitiesLight

Encode special characters in `value` as hexadecimals.

**Return Type:** `string`

## Parameters

| Name      | Type                                           | Optional | Description      |
| :-------- | :--------------------------------------------- | :------- | :--------------- |
| `value`   | `string`                                       | No       | Value to encode. |
| `options` | `import("./core.js").CoreOptions \| undefined` | Yes      | -                |
