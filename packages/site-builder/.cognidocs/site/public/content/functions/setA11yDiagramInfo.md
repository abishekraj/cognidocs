# setA11yDiagramInfo

Add role and aria-roledescription to the svg element.


**Return Type:** `void`

## Parameters
| Name | Type | Optional | Description |
| :--- | :--- | :------- | :---------- |
| `svg` | `D3Element` | No | d3 object that contains the SVG HTML element |
| `diagramType` | `string` | No | diagram name for to the aria-roledescription |
