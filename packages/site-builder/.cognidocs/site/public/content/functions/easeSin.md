# easeSin

Symmetric sinusoidal easing; scales sinIn for t in [0, 0.5] and sinOut for t in [0.5, 1].

**Return Type:** `number`

## Parameters

| Name             | Type     | Optional | Description                                   |
| :--------------- | :------- | :------- | :-------------------------------------------- |
| `normalizedTime` | `number` | No       | Normalized time typically in the range [0, 1] |
