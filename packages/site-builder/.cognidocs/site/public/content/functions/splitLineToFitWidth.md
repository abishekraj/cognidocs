# splitLineToFitWidth

Splits a line into multiple lines that satisfy the checkFit function.

**Return Type:** `MarkdownLine[]`

## Parameters

| Name       | Type               | Optional | Description                    |
| :--------- | :----------------- | :------- | :----------------------------- |
| `line`     | `MarkdownLine`     | No       | Line to split                  |
| `checkFit` | `CheckFitFunction` | No       | Function to check if line fits |
