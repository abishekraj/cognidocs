# scryRenderedComponentsWithType

Finds all instances of components with type equal to `componentClass`.

**Return Type:** `T[]`

## Parameters

| Name   | Type                   | Optional | Description |
| :----- | :--------------------- | :------- | :---------- |
| `root` | `Component<any>`       | No       | -           |
| `type` | `ClassType<any, T, C>` | No       | -           |
