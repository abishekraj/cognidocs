# getClassInfo

Get information about the class fields for this class, given a token processor pointing to the open-brace at the start of the class.

**Return Type:** `ClassInfo`

## Parameters

| Name                  | Type              | Optional | Description |
| :-------------------- | :---------------- | :------- | :---------- |
| `rootTransformer`     | `RootTransformer` | No       | -           |
| `tokens`              | `TokenProcessor`  | No       | -           |
| `nameManager`         | `NameManager`     | No       | -           |
| `disableESTransforms` | `boolean`         | No       | -           |
