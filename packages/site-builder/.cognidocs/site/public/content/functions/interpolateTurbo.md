# interpolateTurbo

Given a number t in the range [0,1], returns the corresponding color from the “turbo” color scheme by Anton Mikhailov.


**Return Type:** `string`

## Parameters
| Name | Type | Optional | Description |
| :--- | :--- | :------- | :---------- |
| `t` | `number` | No | A number in the interval [0, 1]. |
