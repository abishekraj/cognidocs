# allGeneratedPositionsFor

Finds all generated line/column positions of the provided source/line/column source position.

**Return Type:** `GeneratedMapping[]`

## Parameters

| Name     | Type           | Optional | Description |
| :------- | :------------- | :------- | :---------- |
| `map`    | `TraceMap`     | No       | -           |
| `needle` | `SourceNeedle` | No       | -           |
