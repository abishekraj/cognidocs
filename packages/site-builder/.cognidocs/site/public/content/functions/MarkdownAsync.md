# MarkdownAsync

Component to render markdown with support for async plugins through async/await. Components returning promises are supported on the server. For async support on the client, see {@linkcode MarkdownHooks}.


**Return Type:** `Promise<ReactElement>`

## Parameters
| Name | Type | Optional | Description |
| :--- | :--- | :------- | :---------- |
| `options` | `Readonly<Options>` | No | Props. |
