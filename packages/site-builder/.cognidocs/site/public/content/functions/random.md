# random

Generate an array of random bytes collected from hardware noise. `js import { customRandom, random } from 'nanoid' const nanoid = customRandom("abcdef", 5, random) `

**Return Type:** `Uint8Array`

## Parameters

| Name    | Type     | Optional | Description        |
| :------ | :------- | :------- | :----------------- |
| `bytes` | `number` | No       | Size of the array. |
