# toParse5

Transform a hast tree to a `parse5` AST.

**Return Type:** `Parse5Nodes`

## Parameters

| Name      | Type                           | Optional | Description        |
| :-------- | :----------------------------- | :------- | :----------------- |
| `tree`    | `Nodes`                        | No       | Tree to transform. |
| `options` | `Options \| null \| undefined` | Yes      | -                  |
