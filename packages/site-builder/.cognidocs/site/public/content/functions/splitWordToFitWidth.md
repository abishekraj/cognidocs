# splitWordToFitWidth

Splits a word into two parts, the first part fits the width and the remaining part.


**Return Type:** `[MarkdownWord, MarkdownWord]`

## Parameters
| Name | Type | Optional | Description |
| :--- | :--- | :------- | :---------- |
| `checkFit` | `CheckFitFunction` | No | Function to check if word fits |
| `word` | `MarkdownWord` | No | Word to split |
