# presortedDecodedMap

A helper that skips sorting of the input map's mappings array, which can be expensive for larger maps.


**Return Type:** `TraceMap`

## Parameters
| Name | Type | Optional | Description |
| :--- | :--- | :------- | :---------- |
| `map` | `DecodedSourceMap` | No | - |
| `mapUrl` | `string` | Yes | - |
