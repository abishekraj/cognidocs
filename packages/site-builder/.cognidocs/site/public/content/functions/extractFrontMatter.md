# extractFrontMatter

Extract and parse frontmatter from text, if present, and sets appropriate properties in the provided db.

**Return Type:** `FrontMatterResult`

## Parameters

| Name   | Type     | Optional | Description                                |
| :----- | :------- | :------- | :----------------------------------------- |
| `text` | `string` | No       | The text that may have a YAML frontmatter. |
