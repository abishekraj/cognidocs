# context

This is the advanced long-running form of "build" that supports additional features such as watch mode and a local development server. - Works in node: yes - Works in browser: no Documentation: https://esbuild.github.io/api/#build

**Return Type:** `Promise<BuildContext<T>>`

## Parameters

| Name      | Type                         | Optional | Description |
| :-------- | :--------------------------- | :------- | :---------- |
| `options` | `SameShape<BuildOptions, T>` | No       | -           |
