# stateDomId

Create a standard string for the dom ID of an item. If a type is given, insert that before the counter, preceded by the type spacer


**Return Type:** `string`

## Parameters
| Name | Type | Optional | Description |
| :--- | :--- | :------- | :---------- |
| `itemId` | `string` | Yes | - |
| `counter` | `number` | Yes | - |
| `type` | `string \| null` | Yes | - |
| `typeSpacer` | `string` | Yes | - |
