# longestStreak

Get the count of the longest repeating streak of `substring` in `value`.

**Return Type:** `number`

## Parameters

| Name        | Type     | Optional | Description                                     |
| :---------- | :------- | :------- | :---------------------------------------------- |
| `value`     | `string` | No       | Content to search in.                           |
| `substring` | `string` | No       | Substring to look for, typically one character. |
