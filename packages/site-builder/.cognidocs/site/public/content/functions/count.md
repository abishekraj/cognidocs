# count

Returns the number of valid number values (i.e., not null, NaN, or undefined) in the specified iterable; accepts an accessor.

**Return Type:** `number`

## Parameters

| Name       | Type                | Optional | Description  |
| :--------- | :------------------ | :------- | :----------- |
| `iterable` | `Iterable<unknown>` | No       | Input array. |
