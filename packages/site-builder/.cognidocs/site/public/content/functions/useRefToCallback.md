# useRefToCallback

Transforms a given `ref` into `callback`. To transform `callback` into ref use {@link useCallbackRef|useCallbackRef(undefined, callback)}


**Return Type:** `RefCallback<T>`

## Parameters
| Name | Type | Optional | Description |
| :--- | :--- | :------- | :---------- |
| `ref` | `ReactRef<T>` | No | @returns {Function} |
