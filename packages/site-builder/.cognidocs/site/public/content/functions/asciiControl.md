# asciiControl

Check whether a character code is an ASCII control character. An **ASCII control** is a character in the inclusive range U+0000 NULL (NUL) to U+001F (US), or U+007F (DEL).

**Return Type:** `boolean`

## Parameters

| Name   | Type   | Optional | Description |
| :----- | :----- | :------- | :---------- |
| `code` | `Code` | No       | Code.       |
