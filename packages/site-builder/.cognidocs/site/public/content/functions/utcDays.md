# utcDays

This is a convenience alias for utcDay.range(...).

**Return Type:** `Date[]`

## Parameters

| Name    | Type     | Optional | Description                                                                   |
| :------ | :------- | :------- | :---------------------------------------------------------------------------- |
| `start` | `Date`   | No       | A start date object for the range.                                            |
| `stop`  | `Date`   | No       | A stop date object for the range.                                             |
| `step`  | `number` | Yes      | An optional number of steps to apply when calculating the dates in the range. |
