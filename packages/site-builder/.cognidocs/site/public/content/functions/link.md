# link

Returns a new link generator using the specified curve. For example, to visualize links in a tree diagram rooted on the top edge of the display With the default settings the link generator accepts a link object conforming to the DefaultLinkObject interface.


**Return Type:** `Link<any, DefaultLinkObject, [number, number]>`

## Parameters
| Name | Type | Optional | Description |
| :--- | :--- | :------- | :---------- |
| `curve` | `CurveFactory` | No | - |
