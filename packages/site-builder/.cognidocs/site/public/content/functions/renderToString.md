# renderToString

Server-side rendering or rendering to a string Use the `renderToString` function to generate HTML on the server or to generate an HTML string of the rendered math.


**Return Type:** `string`

## Parameters
| Name | Type | Optional | Description |
| :--- | :--- | :------- | :---------- |
| `tex` | `string` | No | A TeX expression. |
| `options` | `KatexOptions` | Yes | An options object. |
