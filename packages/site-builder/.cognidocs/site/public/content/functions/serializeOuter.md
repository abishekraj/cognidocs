# serializeOuter

Serializes an AST element node to an HTML string, including the element node.

**Return Type:** `string`

## Parameters

| Name      | Type                   | Optional | Description            |
| :-------- | :--------------------- | :------- | :--------------------- |
| `node`    | `T['node']`            | No       | Node to serialize.     |
| `options` | `SerializerOptions<T>` | Yes      | Serialization options. |
