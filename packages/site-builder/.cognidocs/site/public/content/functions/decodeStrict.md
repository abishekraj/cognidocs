# decodeStrict

Decodes a string with entities. Does not allow missing trailing semicolons for entities.


**Return Type:** `string`

## Parameters
| Name | Type | Optional | Description |
| :--- | :--- | :------- | :---------- |
| `input` | `string` | No | String to decode. |
| `options` | `DecodingOptions \| EntityLevel` | No | Decoding options. |
