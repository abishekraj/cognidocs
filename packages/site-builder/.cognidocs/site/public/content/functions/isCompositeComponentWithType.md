# isCompositeComponentWithType

Returns `true` if `instance` is a component whose type is of a React `componentClass`.

**Return Type:** `boolean`

## Parameters

| Name       | Type                   | Optional | Description |
| :--------- | :--------------------- | :------- | :---------- |
| `instance` | `ReactInstance`        | No       | -           |
| `type`     | `ClassType<any, T, C>` | No       | -           |
