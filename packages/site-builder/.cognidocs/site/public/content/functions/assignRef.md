# assignRef

Assigns a value for a given ref, no matter of the ref format

**Return Type:** `ReactRef<T>`

## Parameters

| Name    | Type          | Optional | Description                       |
| :------ | :------------ | :------- | :-------------------------------- |
| `ref`   | `ReactRef<T>` | No       | a callback function or ref object |
| `value` | `T \| null`   | No       | a new value                       |
