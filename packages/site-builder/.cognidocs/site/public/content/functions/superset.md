# superset

Returns true if a is a superset of b: if every value in the given iterable b is also in the given iterable a.


**Return Type:** `boolean`

## Parameters
| Name | Type | Optional | Description |
| :--- | :--- | :------- | :---------- |
| `a` | `Iterable<T>` | No | - |
| `b` | `Iterable<T>` | No | - |
