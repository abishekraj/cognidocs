# utcParse

Equivalent to timeParse, except all directives are interpreted as Coordinated Universal Time (UTC) rather than local time. An alias for locale.utcParse (TimeLocaleObject.utcParse) on the default locale.


**Return Type:** `(dateString: string) => Date | null`

## Parameters
| Name | Type | Optional | Description |
| :--- | :--- | :------- | :---------- |
| `specifier` | `string` | No | A specifier string for the date format. |
