# assignMandatoryProperties

**Return Type:** `void`

## Parameters
| Name | Type | Optional | Description |
| :--- | :--- | :------- | :---------- |
| `reflection` | `AstReflection` | No | Reflection object used to gather mandatory properties for the node. |
| `node` | `AstNode` | No | Specified node is modified in place and properties are directly assigned. |
