# isElementOfType

Returns `true` if `element` is a React element whose type is of a React `componentClass`.


**Return Type:** `element is ReactHTMLElement<T>`

## Parameters
| Name | Type | Optional | Description |
| :--- | :--- | :------- | :---------- |
| `element` | `ReactElement` | No | - |
| `type` | `string` | No | - |
