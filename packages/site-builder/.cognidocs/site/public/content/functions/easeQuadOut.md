# easeQuadOut

Reverse quadratic easing; equivalent to 1 - quadIn(1 - t). Also equivalent to polyOut.exponent(2).

**Return Type:** `number`

## Parameters

| Name             | Type     | Optional | Description                                   |
| :--------------- | :------- | :------- | :-------------------------------------------- |
| `normalizedTime` | `number` | No       | Normalized time typically in the range [0, 1] |
