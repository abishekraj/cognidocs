# __rewriteRelativeImportExtension

**Return Type:** `string`

## Parameters
| Name | Type | Optional | Description |
| :--- | :--- | :------- | :---------- |
| `path` | `string` | No | The import specifier. |
| `preserveJsx` | `boolean` | Yes | Causes '*.tsx' to transform to '*.jsx' instead of '*.js'. Should be true when `--jsx` is set to `preserve`. |
