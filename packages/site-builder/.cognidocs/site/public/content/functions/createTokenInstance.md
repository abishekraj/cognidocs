# createTokenInstance

Utility to create Chevrotain IToken "instances" Note that Chevrotain tokens are not real TokenTypes instances and thus the instanceOf cannot be used with them.


**Return Type:** `IToken`

## Parameters
| Name | Type | Optional | Description |
| :--- | :--- | :------- | :---------- |
| `tokType` | `TokenType` | No | - |
| `image` | `string` | No | - |
| `startOffset` | `number` | No | - |
| `endOffset` | `number` | No | - |
| `startLine` | `number` | No | - |
| `endLine` | `number` | No | - |
| `startColumn` | `number` | No | - |
| `endColumn` | `number` | No | - |
