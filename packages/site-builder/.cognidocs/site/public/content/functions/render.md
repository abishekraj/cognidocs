# render

In-browser rendering Call the `render` function with a TeX expression and a DOM element to render into.

**Return Type:** `void`

## Parameters

| Name      | Type           | Optional | Description         |
| :-------- | :------------- | :------- | :------------------ |
| `tex`     | `string`       | No       | A TeX expression.   |
| `element` | `HTMLElement`  | No       | A HTML DOM element. |
| `options` | `KatexOptions` | Yes      | An options object.  |
