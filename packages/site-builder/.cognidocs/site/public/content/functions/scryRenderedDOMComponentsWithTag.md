# scryRenderedDOMComponentsWithTag

Finds all DOM elements of components in the rendered tree that are DOM components with the tag name matching `tagName`.


**Return Type:** `Element[]`

## Parameters
| Name | Type | Optional | Description |
| :--- | :--- | :------- | :---------- |
| `root` | `Component<any>` | No | - |
| `tagName` | `string` | No | - |
