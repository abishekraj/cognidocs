# cont

Checks if the given code point can continue an identifier.

**Return Type:** `boolean`

## Parameters

| Name      | Type                           | Optional | Description          |
| :-------- | :----------------------------- | :------- | :------------------- |
| `code`    | `number \| undefined`          | No       | Code point to check. |
| `options` | `Options \| null \| undefined` | Yes      | -                    |
