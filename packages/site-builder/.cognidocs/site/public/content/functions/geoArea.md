# geoArea

Returns the spherical area of the specified GeoJSON object in steradians. This is the spherical equivalent of path.area.


**Return Type:** `number`

## Parameters
| Name | Type | Optional | Description |
| :--- | :--- | :------- | :---------- |
| `object` | `ExtendedFeature \| ExtendedFeatureCollection \| GeoGeometryObjects \| ExtendedGeometryCollection` | No | - |
