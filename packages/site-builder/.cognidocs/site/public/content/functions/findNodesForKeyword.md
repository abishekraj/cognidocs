# findNodesForKeyword

**Return Type:** `CstNode[]`

## Parameters
| Name | Type | Optional | Description |
| :--- | :--- | :------- | :---------- |
| `node` | `CstNode \| undefined` | No | A CST node in which to look for keywords. If this is undefined, the result is an empty array. |
| `keyword` | `string` | No | A keyword as specified in the grammar. |
