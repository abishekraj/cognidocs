# getIdentifierNames

Get all identifier names in the code, in order, including duplicates.

**Return Type:** `Array<string>`

## Parameters

| Name     | Type           | Optional | Description |
| :------- | :------------- | :------- | :---------- |
| `code`   | `string`       | No       | -           |
| `tokens` | `Array<Token>` | No       | -           |
