# interpolateGreys

Given a number t in the range [0,1], returns the corresponding color from the “Greys” sequential color scheme represented as an RGB string.

**Return Type:** `string`

## Parameters

| Name | Type     | Optional | Description                 |
| :--- | :------- | :------- | :-------------------------- |
| `t`  | `number` | No       | Number in the range [0, 1]. |
