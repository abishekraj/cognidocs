# jsx

**Return Type:** `React.ReactElement`

## Parameters

| Name    | Type                | Optional | Description |
| :------ | :------------------ | :------- | :---------- |
| `type`  | `React.ElementType` | No       | -           |
| `props` | `unknown`           | No       | -           |
| `key`   | `React.Key`         | Yes      | -           |
