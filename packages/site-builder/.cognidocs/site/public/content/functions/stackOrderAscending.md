# stackOrderAscending

Returns a series order such that the smallest series (according to the sum of values) is at the bottom.

**Return Type:** `number[]`

## Parameters

| Name     | Type               | Optional | Description                              |
| :------- | :----------------- | :------- | :--------------------------------------- |
| `series` | `Series<any, any>` | No       | A series generated by a stack generator. |
