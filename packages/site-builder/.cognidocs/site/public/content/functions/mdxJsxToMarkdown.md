# mdxJsxToMarkdown

Create an extension for `mdast-util-to-markdown` to enable MDX JSX. This extension configures `mdast-util-to-markdown` with `options.fences: true` and `options.resourceLink: true` too, do not overwrite them!


**Return Type:** `ToMarkdownExtension`

## Parameters
| Name | Type | Optional | Description |
| :--- | :--- | :------- | :---------- |
| `options` | `ToMarkdownOptions \| null \| undefined` | Yes | - |
