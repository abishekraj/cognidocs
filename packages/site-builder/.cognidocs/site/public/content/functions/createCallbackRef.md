# createCallbackRef

creates a Ref object with on change callback

**Return Type:** `RefObject<T>`

## Parameters

| Name       | Type                                                 | Optional | Description          |
| :--------- | :--------------------------------------------------- | :------- | :------------------- |
| `callback` | `(newValue: T \| null, lastValue: T \| null) => any` | No       | @returns {RefObject} |
