# addNodes

Add nodes to cytoscape instance from provided node array This function processes only the nodes provided in the data structure


**Return Type:** `void`

## Parameters
| Name | Type | Optional | Description |
| :--- | :--- | :------- | :---------- |
| `nodes` | `Node[]` | No | Array of nodes to add |
| `cy` | `cytoscape.Core` | No | The cytoscape instance |
