# easeExp

Symmetric exponential easing; scales expIn for t in [0, 0.5] and expOut for t in [0.5, 1].


**Return Type:** `number`

## Parameters
| Name | Type | Optional | Description |
| :--- | :--- | :------- | :---------- |
| `normalizedTime` | `number` | No | Normalized time typically in the range [0, 1] |
