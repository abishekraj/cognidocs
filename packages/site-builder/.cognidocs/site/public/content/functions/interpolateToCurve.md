# interpolateToCurve

Returns a d3 curve given a curve name

**Return Type:** `CurveFactory`

## Parameters

| Name           | Type                  | Optional | Description                 |
| :------------- | :-------------------- | :------- | :-------------------------- |
| `interpolate`  | `string \| undefined` | No       | The interpolation name      |
| `defaultCurve` | `CurveFactory`        | No       | The default curve to return |
