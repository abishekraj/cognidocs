# timeFormat

Returns a new formatter for the given string specifier. The returned function formats a specified date, returning the corresponding string. An alias for locale.format (TimeLocaleObject.format) on the default locale.


**Return Type:** `(date: Date) => string`

## Parameters
| Name | Type | Optional | Description |
| :--- | :--- | :------- | :---------- |
| `specifier` | `string` | No | A specifier string for the date format. |
