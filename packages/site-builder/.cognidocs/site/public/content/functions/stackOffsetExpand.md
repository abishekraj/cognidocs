# stackOffsetExpand

Applies a zero baseline and normalizes the values for each point such that the topline is always one.


**Return Type:** `void`

## Parameters
| Name | Type | Optional | Description |
| :--- | :--- | :------- | :---------- |
| `series` | `Series<any, any>` | No | A series generated by a stack generator. |
| `order` | `Iterable<number>` | No | An array of numeric indexes representing the stack order. |
