# createSyntaxDiagramsCode

Will generate an html source code (text). This html text will render syntax diagrams for the provided grammar. - See detailed docs for [Syntax Diagrams](http://chevrotain.io/docs/guide/generating_syntax_diagrams.html).


**Return Type:** `string`

## Parameters
| Name | Type | Optional | Description |
| :--- | :--- | :------- | :---------- |
| `grammar` | `ISerializedGast[]` | No | - |
| `config` | `ICreateSyntaxDiagramsConfig` | Yes | - |
