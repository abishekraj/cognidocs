# timeParse

Returns a new parser for the given string specifier. An alias for locale.parse (TimeLocaleObject.parse) on the default locale.


**Return Type:** `(dateString: string) => Date | null`

## Parameters
| Name | Type | Optional | Description |
| :--- | :--- | :------- | :---------- |
| `specifier` | `string` | No | A specifier string for the date format. |
