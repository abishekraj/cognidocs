# generateCirclePoints

**Return Type:** `{
    x: number;
    y: number;
}[]`

## Parameters

| Name         | Type     | Optional | Description                      |
| :----------- | :------- | :------- | :------------------------------- |
| `centerX`    | `number` | No       | x-coordinate of center of circle |
| `centerY`    | `number` | No       | y-coordinate of center of circle |
| `radius`     | `number` | No       | radius of circle                 |
| `numPoints`  | `number` | No       | total points required            |
| `startAngle` | `number` | No       | angle where arc will start       |
| `endAngle`   | `number` | No       | angle where arc will end         |
