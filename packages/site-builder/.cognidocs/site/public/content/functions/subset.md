# subset

Returns true if a is a subset of b: if every value in the given iterable a is also in the given iterable b.


**Return Type:** `boolean`

## Parameters
| Name | Type | Optional | Description |
| :--- | :--- | :------- | :---------- |
| `a` | `Iterable<T>` | No | - |
| `b` | `Iterable<T>` | No | - |
