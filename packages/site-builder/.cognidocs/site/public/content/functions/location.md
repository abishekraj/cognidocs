# location

Create an index of the given document to translate between line/column and offset based positional info. Also implemented in Rust in [`wooorm/markdown-rs`][markdown-rs]. [markdown-rs]: https://github.com/wooorm/markdown-rs/blob/main/src/util/location.rs


**Return Type:** `Location`

## Parameters
| Name | Type | Optional | Description |
| :--- | :--- | :------- | :---------- |
| `file` | `VFile \| Value` | No | File to index. |
