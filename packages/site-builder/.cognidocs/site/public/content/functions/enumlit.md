# enumlit

Defines a literal enum value, such as Direction.Up, specified as enumlit("Direction", "Up").

**Return Type:** `TEnumLiteral`

## Parameters

| Name   | Type     | Optional | Description |
| :----- | :------- | :------- | :---------- |
| `name` | `string` | No       | -           |
| `prop` | `string` | No       | -           |
