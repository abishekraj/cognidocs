# toMarkdown

Turn an mdast syntax tree into markdown.

**Return Type:** `string`

## Parameters

| Name      | Type                           | Optional | Description        |
| :-------- | :----------------------------- | :------- | :----------------- |
| `tree`    | `Nodes`                        | No       | Tree to serialize. |
| `options` | `Options \| null \| undefined` | Yes      | -                  |
