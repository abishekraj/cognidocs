# listItem

Turn an mdast `listItem` node into hast.


**Return Type:** `Element`

## Parameters
| Name | Type | Optional | Description |
| :--- | :--- | :------- | :---------- |
| `state` | `State` | No | Info passed around. |
| `node` | `ListItem` | No | mdast node. |
| `parent` | `Parents \| undefined` | No | Parent of `node`. |
