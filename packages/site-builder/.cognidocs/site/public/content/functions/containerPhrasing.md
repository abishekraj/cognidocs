# containerPhrasing

Serialize the children of a parent that contains phrasing children. These children will be joined flush together.


**Return Type:** `string`

## Parameters
| Name | Type | Optional | Description |
| :--- | :--- | :------- | :---------- |
| `parent` | `PhrasingParents` | No | Parent of flow nodes. |
| `state` | `State` | No | Info passed around about the current state. |
| `info` | `Info` | No | Info on where we are in the document we are generating. |
