# decodeString

Decode markdown strings (which occur in places such as fenced code info strings, destinations, labels, and titles). The “string” content type allows character escapes and -references. This decodes those.

**Return Type:** `string`

## Parameters

| Name    | Type     | Optional | Description      |
| :------ | :------- | :------- | :--------------- |
| `value` | `string` | No       | Value to decode. |
