# fromParse5

Transform a `parse5` AST to hast.


**Return Type:** `Nodes`

## Parameters
| Name | Type | Optional | Description |
| :--- | :--- | :------- | :---------- |
| `tree` | `DefaultTreeAdapterMap["node"]` | No | `parse5` tree to transform. |
| `options` | `Options \| null \| undefined` | Yes | - |
