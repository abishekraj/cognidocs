# ccount

Count how often a character (or substring) is used in a string.

**Return Type:** `number`

## Parameters

| Name        | Type     | Optional | Description                           |
| :---------- | :------- | :------- | :------------------------------------ |
| `value`     | `string` | No       | Value to search in.                   |
| `character` | `string` | No       | Character (or substring) to look for. |
