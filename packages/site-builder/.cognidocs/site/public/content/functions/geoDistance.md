# geoDistance

Returns the great-arc distance in radians between the two points a and b. Each point must be specified as a two-element array [longitude, latitude] in degrees.


**Return Type:** `number`

## Parameters
| Name | Type | Optional | Description |
| :--- | :--- | :------- | :---------- |
| `a` | `[number, number]` | No | Point specified as a two-element array [longitude, latitude] in degrees. |
| `b` | `[number, number]` | No | Point specified as a two-element array [longitude, latitude] in degrees. |
