# __createBinding

**Return Type:** `void`

## Parameters
| Name | Type | Optional | Description |
| :--- | :--- | :------- | :---------- |
| `object` | `object` | No | The local `exports` object. |
| `target` | `object` | No | The object to re-export from. |
| `key` | `PropertyKey` | No | The property key of `target` to re-export. |
| `objectKey` | `PropertyKey` | Yes | The property key to re-export as. Defaults to `key`. |
