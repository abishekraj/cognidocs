# setSourceContent

Adds/removes the content of the source file to the source map.


**Return Type:** `void`

## Parameters
| Name | Type | Optional | Description |
| :--- | :--- | :------- | :---------- |
| `map` | `GenMapping` | No | - |
| `source` | `string` | No | - |
| `content` | `string \| null` | No | - |
