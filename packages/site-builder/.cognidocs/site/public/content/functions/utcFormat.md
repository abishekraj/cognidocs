# utcFormat

Equivalent to timeFormat, except all directives are interpreted as Coordinated Universal Time (UTC) rather than local time. An alias for locale.utcFormat (TimeLocaleObject.utcFormat) on the default locale.


**Return Type:** `(date: Date) => string`

## Parameters
| Name | Type | Optional | Description |
| :--- | :--- | :------- | :---------- |
| `specifier` | `string` | No | A specifier string for the date format. |
