# \_\_runInitializers

**Return Type:** `any`

## Parameters

| Name           | Type         | Optional | Description                                    |
| :------------- | :----------- | :------- | :--------------------------------------------- |
| `thisArg`      | `unknown`    | No       | The `this` argument to use.                    |
| `initializers` | `Function[]` | No       | The array of initializers to evaluate.         |
| `value`        | `any`        | Yes      | The initial value to pass to the initializers. |
