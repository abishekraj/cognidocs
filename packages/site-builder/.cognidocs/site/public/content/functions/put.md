# put

Puts `key` into the backing array, if it is not already present. Returns the index of the `key` in the backing array.

**Return Type:** `number`

## Parameters

| Name     | Type          | Optional | Description |
| :------- | :------------ | :------- | :---------- |
| `setarr` | `SetArray<T>` | No       | -           |
| `key`    | `T`           | No       | -           |
