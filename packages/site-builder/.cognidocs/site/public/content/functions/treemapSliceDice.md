# treemapSliceDice

If the specified node has odd depth, delegates to treemapSlice; otherwise delegates to treemapDice.


**Return Type:** `void`

## Parameters
| Name | Type | Optional | Description |
| :--- | :--- | :------- | :---------- |
| `node` | `HierarchyRectangularNode<any>` | No | - |
| `x0` | `number` | No | - |
| `y0` | `number` | No | - |
| `x1` | `number` | No | - |
| `y1` | `number` | No | - |
