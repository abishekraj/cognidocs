# sourceContentFor

Retrieves the source content for a particular source, if its found. Returns null if not.

**Return Type:** `string | null`

## Parameters

| Name     | Type       | Optional | Description |
| :------- | :--------- | :------- | :---------- |
| `map`    | `TraceMap` | No       | -           |
| `source` | `string`   | No       | -           |
