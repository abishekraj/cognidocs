# stackOrderInsideOut

Returns a series order such that the larger series (according to the sum of values) are on the inside and the smaller series are on the outside. This order is recommended for streamgraphs in conjunction with the wiggle offset. See Stacked Graphsâ€”Geometry & Aesthetics by Byron & Wattenberg for more information.


**Return Type:** `number[]`

## Parameters
| Name | Type | Optional | Description |
| :--- | :--- | :------- | :---------- |
| `series` | `Series<any, any>` | No | A series generated by a stack generator. |
