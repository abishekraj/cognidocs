# MapSource

MapSource represents a single sourcemap, with the ability to trace mappings into its child nodes (which may themselves be SourceMapTrees).


**Return Type:** `MapSource`

## Parameters
| Name | Type | Optional | Description |
| :--- | :--- | :------- | :---------- |
| `map` | `TraceMap` | No | - |
| `sources` | `Sources[]` | No | - |
