# serializeGrammar

Serialize a Grammar to a JSON Object. This can be useful for scenarios requiring exporting the grammar structure for example drawing syntax diagrams.

**Return Type:** `ISerializedGast[]`

## Parameters

| Name       | Type     | Optional | Description |
| :--------- | :------- | :------- | :---------- |
| `topRules` | `Rule[]` | No       | -           |
