# revert

Return the content of a reference without definition as plain text.

**Return Type:** `Array<ElementContent>`

## Parameters

| Name    | Type                        | Optional | Description                   |
| :------ | :-------------------------- | :------- | :---------------------------- |
| `state` | `State`                     | No       | Info passed around.           |
| `node`  | `Extract<Nodes, Reference>` | No       | Reference node (image, link). |
