# @cognidocs/cli

Command-line interface and orchestration for CogniDocs.

## Phase 1: Foundation

This package is part of Phase 1 (Weeks 1-3) and provides:
- CLI argument parsing and command routing
- Configuration file management
- Agent orchestration
- Progress reporting

## Usage

```bash
# Initialize configuration
cognidocs init

# Build documentation
cognidocs build

# Build with watch mode
cognidocs build --watch
```

## Development

```bash
npm run dev     # Watch mode
npm run build   # Build package
npm run test    # Run tests
```
