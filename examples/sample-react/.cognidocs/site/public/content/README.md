# Sample React App

This is a sample React application used for testing CogniDocs parsing and documentation generation.

## Purpose

This project contains:

- React components with TypeScript
- JSDoc comments
- Props interfaces
- Utility functions
- Various documentation patterns

## Usage

```bash
# From the root of CogniDocs
cd examples/sample-react

# Generate documentation for this project
cognidocs init
cognidocs build
```

## Components

- **Button** - A customizable button component with variants
- More components coming in Phase 1...

## Expected Output

When parsed by CogniDocs, this should generate:

- Component API documentation
- Props tables
- Type definitions
- Usage examples
- Function documentation
