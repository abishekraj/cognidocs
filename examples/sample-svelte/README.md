# Sample Svelte App

This is a sample Svelte application for testing CogniDocs Svelte parser functionality.

## Features

- Svelte components with props
- Event dispatching
- Svelte stores (writable, readable, derived)
- Reactive statements
- TypeScript support
- Slots

## Components

- **Button**: A reusable button component with variants
- **Card**: A card component with slots
- **Counter**: A counter component with stores

## Stores

- **count**: A writable store for counter state
- **doubled**: A derived store that doubles the count

## Usage

```bash
# Generate documentation
cognidocs build

# Serve documentation
cognidocs serve
```
